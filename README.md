# 🎤 Voice Gender Detection (Machine Learning)

Proyek ini adalah implementasi sederhana **Machine Learning** untuk mendeteksi apakah sebuah suara termasuk **laki-laki** atau **perempuan** berdasarkan file audio.

Model menggunakan **Random Forest** dengan fitur audio **MFCC (Mel Frequency Cepstral Coefficients)**.

---

## 🚀 Features

- 🎧 Input audio (`.wav`)
- 🔍 Ekstraksi fitur suara (MFCC)
- 🌲 Model Machine Learning (Random Forest)
- 📊 Evaluasi akurasi
- 🔮 Prediksi suara baru
- 🖥️ (Optional) UI sederhana untuk testing

---

1. Audio dibaca menggunakan `librosa`
2. Diubah menjadi fitur numerik (MFCC)
3. Model dilatih menggunakan Random Forest
4. Model digunakan untuk memprediksi audio baru
